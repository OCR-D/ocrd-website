How to Transcribe in Level 3

  1. The transcription is exclusively true to the original. If the graph can be converted with Unicode, this coding is to be used exclusively. Several encodings are not combined to form one graph.
  2. If the graph is not available as standardized coding (Unicode), but via a community standardized coding such as MUFI, this coding is to be used.
  3. If options 1 and 2 are not possible, a code definition shall be used in consultation with the OCR-D Coordination project following the joint agreements reached on major international projects such as IMPACT, EEBO, ECCO.
