Level 3

The diacritics are generally and exclusively regarded as one sign consisting of a basic sign and a diacritic.

If the diacritics exist as a single character, the appropriate code point from the standardized coding tables (Unicode) is used. If no Unicode decoding is available as a single character, the character is mapped as a code point using communitynormalized encodings (MUFI) or encodings defined by the coordination project.

Multiple signs are not combined to form a single sign.
