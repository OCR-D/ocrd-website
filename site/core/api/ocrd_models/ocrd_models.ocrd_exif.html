Technical image metadata

class ocrd_models.ocrd_exif.OcrdExif(img)[source]¶

Represents technical image metadata.

Members: - width / height: pixel dimensions - photometricInterpretation: pixel type/depth, e.g.

‘1’ for b/w, ‘L’ for 8-bit grayscale, ‘RGB’ for 24-bit truecolor, ‘I’ for 32-bit signed integer grayscale, ‘F’ for floating-point grayscale (see PIL concept mode)
