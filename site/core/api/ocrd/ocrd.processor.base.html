ocrd.processor.base module¶

Processor base class and helper functions

class ocrd.processor.base.Processor(workspace, ocrd_tool=None, parameter=None, input_file_grp='INPUT', output_file_grp='OUTPUT', page_id=None, show_help=False, show_version=False, dump_json=False, version=None)[source]¶

Bases: object

A processor runs an algorithm based on the workspace, the mets.xml in the workspace (and the input files defined therein) as well as optional parameter.

add_metadata(pcgts)[source]¶

Adds PAGE-XML MetadataItem describing the processing step

property input_files¶

List the input files.

  • If there’s a PAGE-XML for the page, take it (and forget about all other files for that page)

  • Else if there’s only one image, take it (and forget about all other files for that page)

  • Otherwise raise an error (complaining that only PAGE-XML warrants

    having multiple images for a single page)

(https://github.com/cisocrgroup/ocrd_cis/pull/57#issuecomment-656336593)
