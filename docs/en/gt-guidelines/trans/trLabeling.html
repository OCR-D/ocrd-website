Documentation for gt-labelling : semantic-labelling OCR ground truth data

The automatically generated documentation gives an overview of all components of the OCR-D-GT-labelling scheme.

The documentation of the OCR-D-GT-labelling scheme was generated automatically. The schema is based on the PrimaLab labelling ontology. The documentation was created with the schema documentation tool of the Oxygen editor. The documentation is written in English.
