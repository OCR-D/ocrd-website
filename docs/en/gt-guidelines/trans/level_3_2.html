Level 3

Umlauts are transcribed according to the original, i. e. the umlauts in the common forms Ä, Ö, Ü, ä, ö, ü are distinguished from those which are marked by a superscript e over a vowel. In the case of the superscript e over the vowel, this sign is transcribed by a code point. If there is a specific code point in the respective code table, the standardized coding (Unicode) or the community standardized coding such as MUFI or the OCR-D code definition used in coordination with the OCR-D coordination project shall be used for the coding in subsequent use of jointly concluded agreements of major international projects such as IMPACT, EEBO, ECCO. The Unicode encoding is preferable to other encodings.
