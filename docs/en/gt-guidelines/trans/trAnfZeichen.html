Quotation Marks

The quotation marks in the text are represented with the corresponding Unicode entities to assign them unambiguously to the text (left-facing, right-facing, top or bottom).

Character Entity Description
‘ U+2018 LEFT SINGLE QUOTATION MARK
’ U+2019 RIGHT SINGLE QUOTATION MARK
‚ U+201A SINGLE LOW-9 QUOTATION MARK
‛ U+201B SINGLE HIGH-REVERSED-9 QUOTATION MARK
“ U+201C LEFT DOUBLE QUOTATION MARK
” U+201D RIGHT DOUBLE QUOTATION MARK
„ U+201E DOUBLE LOW-9 QUOTATION MARK
‟ U+201F DOUBLE HIGH-REVERSED-9 QUOTATION MARK
‹ U+2039 SINGLE LEFT-POINTING ANGLE QUOTATION MARK
› U+203A SINGLE RIGHT-POINTING ANGLE QUOTATION MARK
« U+00AB LEFT-POINTING DOUBLE ANGLE QUOTATION MARK
» U+00BB RIGHT-POINTING DOUBLE ANGLE QUOTATION MARK

For the single and double quotation marks, see the Unicode table General Punctuation

For the French quotation marks, see the Unicode table Controls and Latin-1 Supplement
