Typographical Peculiarities

For all element sections (<Word>, <TextLine> and <TextRegion>) the following typographic markup values apply.

Important: The element <TextStyle> is to be used primarily for the documentation of typographical features.
Typographical Particularities <TextStyle> element example
Bold print 
<TextStyle bold="true">
Italic print 
<TextStyle italic="true">
Spaced letters 
<TextStyle letterSpaced="true">
Small capitals 
<TextStyle smallCaps="true">
Printed underlining 
<TextStyle underlined="true">
Strikethrough 
<TextStyle strikethrough="true">
Superscript 
<TextStyle superscript="true">
Subscript 
<TextStyle subscript="true">
