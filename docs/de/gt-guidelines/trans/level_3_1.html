Level 3

Interpreting graphs is completely omitted. The graph is displayed as a code point using standardized encodings (Unicode), communitynormalized codings (MUFI) and encodings defined by the coordination project. A combination of several encodings to form a graph does not take place.

Spaces are only reproduced to separate words from each other.

Punctuation marks are always added to the preceding word.
