Level 3

Eine Interpretation von Graphen wird vollständig unterlassen. Der Graph wird als ein Codepoint unter Nutzung von standardisierten Kodierungen (Unicode), communitynormierten Kodierungen (MUFI) und durch das Koordinierungsgremium festgelegten Kodierungen abgebildet. Eine Kombination von mehreren Kodierungen zu einem Graphen erfolgt nicht.

Die Wiedergabe von Leerzeichen beschränkt sich darauf, dass diese ausschließlich Wörter von einander trennen.

Satzzeichen werden immer an das vorangegangene Wort herangezogen.
