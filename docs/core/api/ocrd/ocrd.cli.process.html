ocrd.cli.process module¶

OCR-D CLI: running task sequences (workflow processing)

ocrd process¶

Process a series of tasks

ocrd process [OPTIONS] TASKS...

Options

-l, --log-level <log_level>¶

Log level

Options:

OFF | ERROR | WARN | INFO | DEBUG | TRACE

-m, --mets <mets>¶

METS to process

-U, --mets-server-url <mets_server_url>¶

TCP host URI or UDS path of METS server

-g, --page-id <page_id>¶

ID(s) of the pages to process

--overwrite¶

Remove output pages/images if they already exist

TASKS¶

Required argument(s)
